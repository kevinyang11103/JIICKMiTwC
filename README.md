# 汽车销售管理系统 java1070

## 项目概述

汽车销售管理系统是一款基于Java开发，使用Springboot和Vue技术栈的前后端分离系统。该系统主要用于汽车销售行业的客户信息管理、车辆信息管理、销售订单处理以及库存财务管理等。

## 技术栈

- **开发语言**：Java
- **数据库**：MySQL
- **前端框架**：Vue
- **后端框架**：Springboot + Mybatis

## 系统版本

- JDK：1.8
- MySQL：8.0.X
- Node.js：12.X.X
- Springboot：2.X.X
- Vue：2.X.X

## 系统角色

系统支持经理和销售两种角色登录管理。

## 系统特点

- **前后端分离架构**：前端界面美观大方，后端提供稳定的数据支持。
- **销售数据分析报表**：提供详实的销售数据分析报表，支持数据统计和图表展示。

## 部署步骤

1. 环境准备：确保JDK、MySQL、Node.js等环境变量配置正确。
2. 数据库部署：导入数据库SQL文件，以构建系统所需数据库。
3. 后端部署：使用IDEA等工具导入后端代码，进行编译并运行。
4. 前端部署：在VS Code中打开前端代码目录，构建并运行。
